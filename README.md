# StudentManagementSystem-WinForms

C# ile geliştirilmiş, Windows Forms tabanlı basit bir öğrenci yönetim sistemidir.  
Uygulama, kullanıcıdan alınan öğrenci bilgilerini DataTable kullanarak listelemeye olanak sağlar.

##  Kullanılan Teknolojiler

- C#
- .NET Framework (WinForms)
- DataTable
- DataGridView

##  Temel Özellikler

- Öğrenci adı ve soyadı ekleme
- Cinsiyet bilgisi seçme (Erkek/Kadın)
- Eklenen öğrencileri tablo halinde görüntüleme
- Basit ve kullanıcı dostu arayüz

## Kurulum

```bash
1. Repoyu klonlayın:
   git clone https://github.com/github_irmakalty/StudentManagementSystem-WinForms.git

2. Visual Studio ile .sln dosyasını açın

3. Projeyi çalıştırın (F5)
```

##  Kullanım

1. Uygulama açıldığında boş bir tablo görüntülenir.
2. Öğrenci adı ve soyadı alanlarını doldurun.
3. Cinsiyet seçimini yapın (Erkek / Kadın).
4. "Ekle" butonuna tıklayın.
5. Öğrenci bilgileri tabloya eklenir.

##  Notlar

- Tüm alanların doldurulması gerekmektedir.
- Eksik bilgi girildiğinde kullanıcı uyarılır.
- Veriler uygulama kapandığında silinir (kalıcı değildir).
